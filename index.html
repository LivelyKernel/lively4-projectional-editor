About

The Lively Projectional Editor component is in the 
templates
folder of this repository. This component implements most of the functionality and is imported as soon as this page is opened. Some functionality, such as the parsing of Babel AST nodes into Blockly blocks is implemented in seperate files (
babel_tools.js
and 
blockly_tools.js
) in the 
lib
folder. This folder also contains compressed copies of Blockly and Babel 6, which are automatically loaded.

Notes on Blockly

The current version of Blockly has some problems when it is used in web components. The version included with this repository has been modified to prevent an (incorrect) error message after Blockly was loaded.
